Linux 下帝国时代： 0 A.D

Apr 2nd, 2012 | Comments

0 A.d 是一款类似帝国时代的开源即时战略游戏。 这个游戏以公元前 500 年至公元 500 年之间为背景，包括了六个独特的种族文明。

特色：3D，画面古典，建筑可换角度，英文

 

 

 

在 Ubuntu 下的安装方法：

sudo add-apt-repository ppa:wfg/0ad 
sudo apt-get update 
sudo apt-get install 0ad

源代码下载

http://wildfiregames.com/0ad/

http://zh.wikipedia.org/wiki/0_A.D.
